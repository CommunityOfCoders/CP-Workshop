* Modular Arithmetic
```cpp
Addition
(a + b) % m = ((a % m) + (b % m)) % m

Subraction
(a - b) % m = ((a % m) - (b % m)) % m

Multiplication
(a * b) % m = ((a % m) * (b % m)) % m

Division (We need to use Mod Inverse)
(a / b) % m != ((a % m) / (b % m)) % m
```
